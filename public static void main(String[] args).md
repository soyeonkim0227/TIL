## public static void main(String[] args)의 의미

- #### public

  public을 포함한 접근 제한자에는 protected, default, private이 있다.

  - public은 다른 클래스, 다른 패키지 모든 곳에서 접근이 가능하다.
  - protected는 같은 패키지 내에서만 접근이 가능하고, 다른 패키지일 경우 해당 클래스를 상속받은 경우에 접근이 가능하다.
